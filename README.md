# 医院管理系统简介 python140

## 项目概述

医院管理系统是一个基于Python和Django框架开发的应用，旨在为医院提供一套完整的信息化解决方案。系统主要涵盖患者、医生和管理员三个角色，以实现高效的医疗流程管理和资源协调。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：HTML

## 功能模块

### 管理员

- 科室管理页面：展示科室列表，不涉及修改科室信息。
- 医生管理页面：管理医生信息，包括查看医生列表和添加新医生。
- 药品管理页面：维护药品信息，可查看和添加新药品。
- 患者管理页面：管理患者信息，包括查看患者列表和添加新患者。

### 医生

- 接诊房间页面：查看待诊患者列表，只能接诊所属科室的患者，完成就诊并输入医嘱、开药。
- 就诊记录页面：查看历史就诊记录和药品清单。

## 系统角色

- 患者：接受医疗服务的一方。
- 医生：提供医疗服务，包括接诊和开药。
- 管理员：负责系统的日常维护，管理医生、患者和药品信息。

## 运行环境

- Python环境
- Django框架
- MySQL数据库
- 支持HTML的Web浏览器

## 部署步骤

1. 安装Python环境和Django框架。
2. 配置MySQL数据库。
3. 克隆源代码到本地。
4. 执行数据库迁移命令，建立数据模型。
5. 运行Django应用。
